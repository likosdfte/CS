2. Formateando un Texto Web

Aquí vemos las etiquetas que modifican el texto.

Etiquetas de modificación de significado

  • Este texto es importante (Etiqueta strong).
  • Y este texto tiene énfasis (Etiqueta em).

Etiquetas orientadas al código

Para mostrar código de programación, usamos la etiqueta <p>Hola Mundo</p>.

Etiquetas de listas

La lista de arriba (con números) usa la etiqueta ol. La lista de navegación (sin números) usa la etiqueta ul.
